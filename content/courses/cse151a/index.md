## Course Overview

**CSE 151A: Machine Learning - Learning Algorithms** is an undergraduate course in the Computer Science and Engineering Department at UC San Diego.

## Topics Covered

- Supervised learning algorithms
- Decision trees and random forests
- Support vector machines
- Neural networks
- Deep learning fundamentals
- Model evaluation and validation
