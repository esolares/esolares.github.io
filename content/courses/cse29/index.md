## Course Overview

**CSE 29: Systems Programming and Software Tools** is an undergraduate course in the Computer Science and Engineering Department at UC San Diego.

## Topics Covered

- Unix/Linux command line
- Shell scripting
- Version control (Git)
- Build systems and automation
- Debugging tools
- Systems programming in C
